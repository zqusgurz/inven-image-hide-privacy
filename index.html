개인정보처리방침

본 확장 프로그램은 사용자 정보를 절대로 수집, 저장, 전송하지 않습니다. 이 확장 프로그램은 브라우저 내에서 실행되는 단순한 이미지 표시 제어 기능만 제공하며, 외부 서버와 통신하거나 개인 정보를 처리하지 않습니다.

1. 수집하는 정보

본 확장 프로그램은 사용자의 이름, 이메일, 계정 정보, 브라우저 데이터, 방문 기록 등 어떠한 개인정보도 수집하지 않습니다.

2. 데이터 처리 방식

확장의 모든 기능은 사용자의 브라우저 로컬 환경에서만 처리되며, 어떠한 정보도 외부로 전송되지 않습니다.

3. 제3자 제공

개인 데이터를 수집하지 않으므로 제3자에게 제공하거나 공유할 정보가 없습니다.

4. 외부 서비스 사용

본 확장 프로그램은 광고, 분석(Analytics), 추적 도구, 쿠키 등 어떠한 외부 서비스도 사용하지 않습니다.

5. 보안

개인 정보를 취급하지 않기 때문에 개인정보 유출이나 보안 사고가 발생할 여지가 없습니다. 확장 기능은 안전하게 브라우저 내부에서만 수행됩니다.
